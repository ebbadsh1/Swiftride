# 🚗 SwiftRide Analytics Dashboard
An interactive Data Science dashboard built with Python and Streamlit to analyze ride-hailing trends and predict fares.

## 🤖 Model Performance
I trained a Random Forest model that achieves an **R² score of 0.9973**, meaning it predicts fares with incredible accuracy.
