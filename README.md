# 🚗 SwiftRide Analytics Dashboard
An interactive Data Science dashboard built with Python and Streamlit to analyze ride-hailing trends and predict fares.

## 📊 Project Overview
This project provides a comprehensive look into fleet performance and pricing. 

* **Executive Overview:** High-level KPIs including revenue and trip volume.
* **Trip Analytics:** Deep dive into vehicle types and distance vs. fare trends.
* **Driver Performance:** Rating distributions across major cities.
* **ML Fare Predictor:** Real-time fare estimation with high precision.

## 🤖 Model Performance
I trained a Random Forest model that achieves an **R² score of 0.9973**, meaning it predicts fares with incredible accuracy.

| Metric | Value |
| :--- | :--- |
| **R² Score** | 0.9973 |
| **MAE** | 13 PKR |
| **RMSE** | 22 PKR |
