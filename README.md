# 🚗 SwiftRide Analytics Dashboard
An interactive Data Science dashboard built with Python and Streamlit to analyze ride-hailing trends and predict fares.

## 🤖 Project Highlights
* **High-Precision Forecasting:** Trained a Random Forest model with an **R² score of 0.9973**.
* **Comprehensive Analytics:** Visualized 7,000+ trip records across major cities including Karachi, Lahore, and Peshawar.
* **Full-Stack Implementation:** Integrated a SQL backend for data management and Streamlit for the user interface.

## 📊 Model Performance Metrics
| Metric | Value |
| :--- | :--- |
| **R² Score** | 0.9973 |
| **Mean Absolute Error (MAE)** | 13 PKR |
| **Root Mean Square Error (RMSE)** | 22 PKR |

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Plotly, Streamlit
* **Database:** SQLite
